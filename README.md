# **Pre-requisite:**
  1. Download [Github Desktop](https://desktop.github.com/download/)
  2. Download [Git Large File Storage](https://git-lfs.com)


## **Fork this Repo First!!!**

### To know if it's working:
1. Click **Fork** Repository
2. Open **Github Desktop**
3. Click Clone your **Forked** Repository, *not this repo*

> Make sure that it is your forked repo not mine, the link should look like "github.com/<<your_github_username>>/Game-Development

4. Copy your forked repo and paste it there
> Optional: Choose a path to your desired location/directory *(Mine is in /Documents/Code/Unity/Game-Development)*


5. Open **Unity Hub**
6. Click "Add" > "Add project from disk"
7. Click the **Game-Development** from Project
8. Then happy coding!
###### lala


### To update your fork repo:
1. Open Github Desktop
2. You can see the changes and history
3. Fill the Summary text field to commit
> Optional: Add descriptions to know what's the update on your end

4. **Push** in your fork
5. **ALWAYS** sync/pull your latest commit
> There can be error the next time you commit after your push

### To Sync your Fork Repo to the Main Repo
1. Go to your repo link, your fork
2. You should be able to find the sync fork
